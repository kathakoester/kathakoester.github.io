Let's Getting Started

Welcome to zemo - React Personal Portfolio Template.

This guide assumes basic knowledge of React, HTML, CSS and Javascript.

All of the template files will be delivered in a zip file. Simply extract the zip file, import it to your favorite text editor and it's ready for your customizations.

If you had any problems, please do not hesitate to contact us from sage.kv4@gmail.com

Requirements

Make sure in your system NPM and node is installed in your system.

node and NPM 

                        
                            node install
                            npm install

Install NPM dependencies

First you need to install all the npm dependencies in your system using below command. 

                        
                            npm install

Development server

To run development server you need to run below command and that will redirect you to http://localhost:3000/ and you changes will automatically reflect there. 

                        
                            npm run start

What's Included
